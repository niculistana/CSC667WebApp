# Liars Dice
Multiplayer Dice Player Game.

### Requirements
```
ruby 2.5.8
rails 5.2.4
```

### Development
```
bundle
rake db:migrate
rails assets:precompile
rails s
```

### Note
This project is currently undergoing migration. Most methods may not necessarily be up to date.
